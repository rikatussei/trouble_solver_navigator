概要

禁酒をサポートするアプリケーションです。
記録や目標設定を行うことで、禁酒を達成しやすくする機能を提供します。

目的

禁酒に取り組む人々を支援します。
達成感を感じやすい仕組みでモチベーションを維持します。
アルコール消費に関する習慣改善をサポートします。
使用技術

フロントエンド：HTML, CSS, JavaScript
バックエンド：Java
データベース：PostgreSQL
インフラ：Docker（予定） or AWS
セットアップ手順
